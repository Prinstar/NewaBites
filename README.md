# 🍔 Food Ordering App

> An Android food ordering application built entirely in Kotlin using XML layouts.
> Developed as a school project and portfolio piece to demonstrate Android development fundamentals and UI design.

---

## 📚 Project Overview

This application allows users to browse a food menu, add items to a cart, and view an order summary.
The project focuses on frontend Android development, UI design, and implementing business logic without a database.

---

## ✨ Features

* 🥗 Browse food menu items
* 🛒 Add and remove items from cart
* 💰 Dynamic total price calculation
* 📦 Order summary screen
* 📱 Clean and responsive XML-based layouts
* 🎨 Material Design components

---

## 🛠 Tech Stack

* **Language:** Kotlin
* **UI Design:** XML Layouts
* **IDE:** Android Studio
* **Build System:** Gradle
* **Minimum SDK:** 21+
* **Architecture:** Activity-based structure

---

## 🧠 Concepts Demonstrated

This project demonstrates:

* Kotlin programming fundamentals
* Android Activity lifecycle
* Intent navigation between screens
* RecyclerView implementation (if used)

* UI layout design with XML
* Basic shopping cart logic
* Clean project structure
